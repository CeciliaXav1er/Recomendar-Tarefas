# Recomendar-Tarefas
Este projeto é uma aplicação de console desenvolvida em C para auxiliar estudantes de programação a praticarem tópicos específicos. O sistema funciona como um gerenciador de banco de questões que utiliza um algoritmo de filtragem para recomendar o "próximo exercício" ideal para o usuário.

**Principais Diferenciais:**
  Recomendação Inteligente: Filtra exercícios por nível de dificuldade (1-5) e por tags técnicas (ex: Alocação Dinâmica, Manipulação de Arquivos, Recursão).
  
  Gestão de Dados (CRUD): Permite ao administrador cadastrar, listar, buscar e excluir tanto tarefas quanto usuários.
  
  Histórico de Progresso: O sistema memoriza quais atividades o usuário já realizou para garantir que as recomendações sejam sempre inéditas.
  
  Persistência Híbrida: Utiliza arquivos binários (.dat) para alta performance e segurança no armazenamento local, e exporta relatórios em .csv para análise externa.
